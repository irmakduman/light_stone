# Light Stone

## About

Light Stone is an action-based web game created as a graduation project.

The project was built using HTML, CSS and JavaScript.

The game idea, story and gameplay were planned by our team.

AI-assisted tools were used during development. We reviewed, tested and improved the final project.

## Features

- Action-based gameplay
- Story progression
- Player choices
- Multiple scenes

## Technologies

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project.
2. Open the `index.html` file in your web browser.

## Project Status

Completed as a graduation project.
